Getting Started

1. Enter your Oracle Fusion instance URL and credentials

2. Click "Connect" to establish connection

3. Use the navigation menu to access different features

Features

  • User Details: Get comprehensive user information
  • Role Management: Assign and remove user roles
  • Data Security: Manage data security contexts
  • Areas of Responsibility: Assign AORs to users
  • Bulk Operations: Process multiple records at once
  • Excel Upload: Upload Excel files for bulk processing

Excel Templates

Download the appropriate template for your bulk operation:

  • Role Assignment: username, role_name
  • Data Security: username, role_name, security_context, security_value
  • AOR Assignment: username, aor_name, aor_type
